## Task 2: Independent Verification of Singh's delta^2 = 3/8

**Script:** `src/experiments/singh_delta_verification.py`
**Results:** `src/experiments/results/singh_delta_verification.json`
**Engine:** Python 3.14 (numpy, with custom octonion multiplication)

### The Claim

Singh (2025, arXiv:2508.10131) claims that for elements of J3(O) -- 3x3 Hermitian
octonionic matrices -- with equal diagonal entries a = b = c = s and octonionic
off-diagonal entries normalized to |x|^2 = |y|^2 = |z|^2 = 1/8, the eigenvalue
spread is delta^2 = 3/8, independent of the specific octonionic entries.

### Analytic Derivation

For an equal-diagonal J3(O) element with |x|^2 = |y|^2 = |z|^2 = N:

1. Trace = 3s, S2 = 3s^2 - 3N, det = s^3 - 3sN + 2*Re(x(yz))

2. Mean-subtracted eigenvalues mu_i = lambda_i - s satisfy the reduced cubic:
   mu^3 - 3N*mu + 2*Re(x(yz)) = 0

3. By Vieta's formulas:
   - Sum(mu_i) = 0
   - Sum(mu_i * mu_j) = -3N (where sum is over i < j)
   - Product(mu_i) = -2*Re(x(yz))

4. Therefore: Sum(mu_i^2) = -2 * Sum(mu_i * mu_j) = 6N

5. Defining delta^2 = (1/2) * Sum(mu_i^2) = 3N

6. For N = 1/8: **delta^2 = 3/8**.

**The critical insight:** delta^2 depends ONLY on the normalization N = |x|^2, via
Vieta's formula for the p-coefficient of the reduced cubic. It does NOT depend on the
q-coefficient (which involves Re(x(yz))). The specific octonionic entries are irrelevant
to the eigenvalue spread.

### Numerical Verification

**Test 1: Singh's normalization (N = 1/8, 10,000 random J3(O) elements)**

| Quantity | Value |
|----------|-------|
| Mean delta^2 | 0.37500000 |
| Predicted | 0.37500000 |
| Std deviation | 7.80 x 10^-16 |
| Deviation from 3/8 | 3.89 x 10^-16 |

**delta^2 = 3/8 is EXACT to machine precision.** This is not approximate -- it follows
from the Vieta formula analytically and is confirmed numerically to 15 decimal places.

**Test 2: Normalization dependence (1,000 trials each)**

| N | Mean delta^2 | Predicted 3N | Ratio |
|---|-------------|-------------|-------|
| 1/8 | 0.375000 | 0.375000 | 1.000000 |
| 1/4 | 0.750000 | 0.750000 | 1.000000 |
| 1/2 | 1.500000 | 1.500000 | 1.000000 |
| 1 | 3.000000 | 3.000000 | 1.000000 |
| 2 | 6.000000 | 6.000000 | 1.000000 |

**The formula delta^2 = 3N holds exactly for all normalizations.** The value 3/8 is
NOT special -- it simply reflects the choice N = 1/8. Different normalizations give
different delta^2.

**Test 3: Unequal diagonal entries (a != b != c)**

For 5,000 elements with random diagonal entries [0.5, 2.0] and N = 1/8:
- Mean delta^2 = 0.5609 (NOT 0.375)
- Range: [0.375, 1.118]
- delta^2 = 3/8 is the MINIMUM (achieved when a = b = c)

**delta^2 = 3/8 holds ONLY for equal diagonal entries.** Unequal diagonals add
to the spread.

**Test 4: Re(x(yz)) effect on individual eigenvalues**

For 1,000 elements with a = b = c = 1, N = 1/8:
- delta^2 = 0.375 exactly for ALL elements (std = 7.6 x 10^-16)
- Re(x(yz)) range: [-0.037, 0.033]
- mu_1/mu_3 ratio range: [-1.487, -0.714]

**Re(x(yz)) changes individual eigenvalues but NOT the total spread.**
The eigenvalue PATTERN (which is largest, ratio between them) depends on
Re(x(yz)), which varies with the octonionic entries.

### Assessment of Singh's Framework

**What is genuinely algebraic:**
- delta^2 = 3N for equal-diagonal J3(O) elements (exact Vieta identity)
- For N = 1/8: delta^2 = 3/8 (algebraic consequence)

**What is assumed (not derived from J3(O)):**
- The normalization N = 1/8 (a convention choice)
- The trace split Tr(X_l) : Tr(X_u) : Tr(X_d) = 1 : 2 : 3 (free parameter)
- Equal diagonal entries a = b = c (special case, not generic)
- The "minimality principle" for CG ladder (physical postulate)

**What is determined by free input:**
- Individual eigenvalue ratios (depend on Re(x(yz)), which is not fixed)
- Mass hierarchies between sectors (depend on trace split)
- First-generation mass ratios (direct consequence of 1:2:3 assumption)

**Verdict: CONFIRMED that delta^2 = 3/8 is an exact algebraic result for the specified
class of J3(O) elements. However, the result is less remarkable than presented: it is
a Vieta identity (delta^2 = 3N) applied at a specific normalization (N = 1/8). The
normalization is a choice, and the mass ratio predictions require additional assumptions
(trace split, equal diagonal, minimality) that are not derived from J3(O).**
